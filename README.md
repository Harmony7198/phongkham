# PHÒNG KHÁM MINI – TIẾP NHẬN BỆNH NHÂN VÀ KÊ ĐƠN THUỐC

Ứng dụng web thực hành xây dựng bằng **HTML5, CSS3 và JavaScript ES Module**.

---

# Mục tiêu

- Thực hành HTML5
- Thực hành CSS3
- Thực hành JavaScript ES Module
- Thực hành LocalStorage
- Unit Test với Vitest
- Business Test với Vitest
- UI Test với Playwright
- CI/CD với GitHub Actions
- Triển khai GitHub Pages

# Cài đặt

## Clone dự án

```bash
git clone <repository-url>
```

## Cài đặt thư viện

```bash
npm install
```

---

# Chạy ứng dụng

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Preview cho Playwright

```bash
npm run preview:test
```

---

# Kiểm thử

## Chạy tất cả

```bash
npm run test:all
```

## Unit Test

```bash
npm run test:unit
```

## Business Test

```bash
npm run test:business
```

## Coverage

```bash
npm run test:coverage
```

## Playwright

```bash
npm run test:e2e
```

## Playwright UI

```bash
npm run test:e2e:ui
```

## Playwright Report

```bash
npm run test:e2e:report
```
